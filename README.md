# 🌐 CS50 Network – Social Media App

A **social media web application** built with Django that allows users to create posts, follow other users, like posts, and browse a personalized feed.  
This project was developed as part of **CS50’s Web Programming with Python and JavaScript**.

---

## 🚀 Features
- 📝 Create, edit, and delete posts  
- ❤️ Like and unlike posts  
- 👥 Follow and unfollow users  
- 📰 Personalized feed showing followed users’ posts  
- 📄 Pagination for smooth browsing  
- 🔐 User authentication (register, login, logout)

---

## 🛠 Tech Stack
- Django (Python)  
- JavaScript (AJAX)  
- HTML5, CSS3  
- SQLite (default DB)

---

## 📦 Installation

Run the following commands:

```bash
# Clone the repository
git clone https://github.com/pgatundu/Networking.git

# Navigate into the project
cd Networking

# Install dependencies
pip install -r requirements.txt

# Apply migrations
python manage.py migrate

# Start the development server
python manage.py runserver
